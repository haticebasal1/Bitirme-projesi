📱 PhoneCase - E-Commerce Graduation Project
Bu proje, .NET 8 platformu üzerinde MVC (Model-View-Controller) mimarisi kullanılarak geliştirilmiş modern bir e-ticaret (telefon kılıfı satışı) web uygulamasıdır. Kullanıcıların alışveriş deneyimini en üst seviyeye çıkarmak ve yöneticilerin sistemi kolayca kontrol etmesini sağlamak amacıyla iki ana bölümden (Kullanıcı Arayüzü ve Yönetim Paneli) oluşmaktadır.

🚀 Özellikler (Features)

👤 Kullanıcı Arayüzü (Frontend / User Interface)

📍 Kullanıcı Kayıt ve Giriş Sistemi: Güvenli kimlik doğrulama mekanizması.

📍 Dinamik Ana Sayfa: Slider (slayt) yapısında öne çıkan ürünler, popüler ürünler ve tüm ürünlerin listelenmesi.

📍 Detaylı Ürün Sayfası: Ürün açıklamaları, stok durumları, fiyat bilgileri ve kategori bazlı filtreleme.

📍 Favori Sistemi: Beğenilen ürünlerin favorilere eklenmesi, listelenmesi ve sepetle senkronize çalışması.

📍 Gelişmiş Sepet ve Sipariş Akışı: Sepete ürün ekleme/çıkarma, adet güncelleme, fatura/adres bilgilerinin girilmesi ve sipariş tamamlama simülasyonu.

👑 Yönetim Paneli (Admin Panel)
📍 Özet Gösterge Paneli (Dashboard): Aktif ürün sayısı, kategori sayısı, toplam sipariş miktarı gibi istatistiklerin grafiksel ve sayısal takibi.

📍 Kategori Yönetimi: Yeni kategori oluşturma, güncelleme, silme ve resim yükleme işlemleri.

📍 Gelişmiş Ürün Yönetimi (CRUD): Ürün listesi, pasif/silinmiş ürünleri görebileceğiniz Çöp Kutusu (Soft Delete) mekanizması.

📍 Sipariş Yönetimi: Tüm siparişlerin durumunu (Hazırlanıyor, Kargoya Verildi, Teslim Edildi) tek tıkla güncelleme ve filtreleme.

🛠️ Teknolojiler (Technologies Used)

📍 Backend: .NET 8 (C#)

📍 Architecture: ASP.NET Core MVC (Model-View-Controller)

📍 Database & ORM: Entity Framework Core

📍 Frontend: HTML5, CSS3, Bootstrap 5, JavaScript

📍 UI Template: Colorlib & ThemeWagon (Özelleştirilmiş admin ve kullanıcı temaları)
